## 简介

一个Telegram Covid-19 Bot。

![nCovBot](https://github.com/varz1/pics/blob/master/bot.png?raw=true)

## 主要功能

- 支持地区列表
- 国内疫情概览（包括中国疫情地图
- 本土疫情趋势图
- 最新新闻（返回新闻概览以及列表
- 中高风险地区查询
- 输入地区名称返回疫情数据
